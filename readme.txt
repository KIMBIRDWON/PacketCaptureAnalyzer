[환경]
- OS: Ubuntu 22.04 이상
- Python 3, Tkinter, pip, scapy 필요

[설치 방법]

터미널에서 다음 명령을 순서대로 실행합니다.

1) 파이썬 및 Tkinter, pip 설치
   sudo apt install python3 python3-tk python3-pip

2) scapy 설치
   pip3 install scapy

[실행 방법]

1) 소스 파일이 있는 디렉터리로 이동
   cd 패킷_프로그램_폴더

2) 프로그램 실행
   sudo python3 PacketCaptureAnalyzer.py
